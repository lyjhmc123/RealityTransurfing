# Project Summary (CRITICAL)

## 프로젝트 개요

**Intertext Magazine Issue No.1 - Reality Transurfing**은 트랜서핑 시리즈가 담고 있는 개념 및 세계관을 깊이있게 전달하는 디지털 매거진입니다.

## 목표 (Goals)

1. **트랜서핑 세계관 전달** - 트랜서핑 시리즈가 담고 있는 개념 및 세계관을 깊이있게 전달
2. **전시장 경험** - 전시장에 입장하고, 섹션별로 이동하여 감상한 후 퇴장하는 것처럼 컨텐츠를 seamless하게 볼 수 있는 경험 (스크롤 기반)
3. **매거진 리딩 경험** - 오프라인 매거진을 보듯 커버스토리 → 타이틀 → 리드문 → 본문을 읽는 경험 제공
4. **댓글 기능** - 각 아티클별로 비로그인 기반 댓글을 작성하고 수정할 수 있는 기능 제공

## 비범위 (Non-Goals)

- 로그인
- 댓글 쓰기/편집 외의 CRUD 기능

## 타깃 사용자

- 트랜서핑 시리즈에 관심이 있는 사람
- 트랜서핑은 모르지만 영성, 심리학, 뉴에이지 철학 등에 관심이 있는 사람

## 핵심 가치 제안

트랜서핑에서 나오는 주요개념을 이해하고, 세계관을 경험할 수 있는 컨텐츠 제공

## 기술 스택

- MUI 7.x (Material UI)
- Vite 7.x
- Storybook 10.x

## 중요 규칙

### 1. 컴포넌트 작성

- 모든 UI 컴포넌트는 MUI 기반으로 작성
- 스타일링은 MUI의 `sx` prop 사용
- 컴포넌트는 독립적이고 재사용 가능하게 설계

### 2. 스토리 작성

- 모든 컴포넌트는 Storybook 스토리와 함께 작성
- 디자이너가 이해하기 쉬운 명확한 설명 포함
- Props 변형을 시각적으로 확인할 수 있도록 구성

### 3. 디자인 시스템

- 색상, 타이포그래피는 테마 파일에서 중앙 관리
- 일관된 spacing, elevation, borderRadius 적용
- Style 섹션에서 디자인 토큰 문서화

### 4. 작업 분리 원칙

- **UI 레이어**: 순수 프레젠테이션 컴포넌트 (로직 없음)
- **로직 레이어**: 비즈니스 로직, 상태 관리, API 호출
- Storybook에서는 UI 레이어만 다룸
